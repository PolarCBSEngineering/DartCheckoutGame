# PWA Icons

Bu klasöre aşağıdaki icon dosyaları eklenmelidir:

- `icon-192.png` - 192x192 piksel
- `icon-512.png` - 512x512 piksel  
- `apple-touch-icon.png` - 180x180 piksel (iOS için)

## Icon Tasarımı

Icon'lar dart tahtası veya dart oku temalı olabilir. Basit bir tasarım için:
- Arka plan: #2c3e50 (koyu mavi-gri)
- Vurgu: #f1c40f (altın sarısı)
- Dart tahtası veya dart oku silüeti

## Online Icon Generator

Ücretsiz icon oluşturmak için:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Geçici Çözüm

Şimdilik icon dosyaları olmadan da uygulama çalışır, ancak PWA install özelliği tam çalışmayabilir.
